# secretsifter-extension
Live secrets and credentials scanner for Chrome and Firefox. Detects API keys, tokens, passwords, and private keys in real-time network traffic across JS files, JSON APIs, XML responses, HTML pages, and WebSocket frames. Designed for penetration testers, bug bounty hunters, and security engineers.

## Install
- [Firefox](https://addons.mozilla.org/en-US/firefox/addon/secretsifter-extension/)
- [Chrome](https://chromewebstore.google.com/detail/secretsifter-live-credent/okjoofmgkkgopigaipccailblmpcchjo)
